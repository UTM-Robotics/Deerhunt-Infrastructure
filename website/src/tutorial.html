Tutorial

Overview

The game is asynchronous turn-based with two players competing against each other.
The map is a 40x40 grid, with a few maps in rotation. Each map is symmetrical.
There is an odd number of resource nodes on the map.
There is fog of war which will restrict vision for both players.
Each game lasts for 100 rounds or if one player's units are eliminated.

Objectives

There are two ways to win the game.
1. Eliminate all of the enemy's units.
2. Have more resources than the enemy by 100 turns. If resources are tied, Player 2 wins.

Units

There are two types of units available: Melees and Workers.
Melee units are able to attack other units and can duplicate itself to create more Melee units.
Worker units are able to mine resources from resource nodes, they cannot duplicate itself.
Players will start off with a few of each units.

Information

Every turn, players will be given basic information about the state of the game.
game_map is the grid which represents the current map.
your_units is the list of every unit that belongs to you.
enemy_units is the list of every unit that your units can see (fog of war) that belong to the enemy.
resources is the number of resources that you currently possess.
turns_left is the number of turns that remain until the game ends.

Commands

Append commands for units to the moves list to perform actions.
Commands can be found in helper_classes.py, there is documentation for every command.
Use your_units.get_all_unit_of_type('worker') to get a list of all of your worker units.
Use your_units.get_all_unit_of_type('melee') to get a list of all of your melee units.

Moving

There are four directions that each unit can move in: UP, DOWN, LEFT, RIGHT
Each unit can only move 1 space at time.
Use unit.move(direction) to move the indicated unit.
Use map.bfs(start, end) to get the shortest path from start to end.

Attacking

Melee units can attack a unit from 1 cardinal range away.
All units need only one attack to be killed.
Use unit.attack(direction) to perform an attack.

Mining

Worker units can mine a resource node if it is on top of one.
Resource nodes have a limited amount of resources.
It takes 5 turns to mine from a node. During this time, the worker is locked in place.
Use unit.mine() to begin mining for the indicated unit.
Use map.closest_resources(unit) to find the closest resource node from the unit.

Duplicating

Melee units can duplicate itself to create another melee unit.
Duplication costs 75 resources and take 5 turns to complete. During this time, the melee is locked in place.
Use unit.duplicate(direction) to create a unit in the direction indicated.

Good luck and have fun!
